🚀 Primeira Versão do Projeto

Salvando a primeira versão de um projeto no GitHub através do Git na máquina.

1. Inicializando o Git

O projeto foi inicializado com git init, permitindo o controle de versões localmente.

2. Primeiro Commit

Os arquivos foram adicionados com git add . e salvos com o primeiro git commit.

3. Enviando para o GitHub

Após conectar o repositório remoto, a primeira versão foi enviada para o GitHub utilizando git push.
